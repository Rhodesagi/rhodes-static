# Desktop Calculator

A clean, modern desktop calculator built with pywebview.

## Features

- Beautiful dark-themed UI
- Full keyboard support (numbers, operators, Enter, Escape, Backspace)
- Calculation history (last 5 calculations)
- Responsive button animations
- Error handling (division by zero)

## Installation

```bash
pip install -r requirements.txt
```

## Usage

```bash
python main.py
```

Or open `calculator.html` directly in any browser.

## Keyboard Shortcuts

- `0-9` - Input numbers
- `.` - Decimal point
- `+ - * /` - Operators
- `Enter` or `=` - Calculate
- `Escape` - Clear all (AC)
- `Backspace` - Delete last digit
