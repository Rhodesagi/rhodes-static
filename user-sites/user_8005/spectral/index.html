Spectral Coloring: Register Allocation

ALGORITHM VERIFIED | Lev Osin | April 2026 | Alcor Scottsdale

A register allocation algorithm using spectral graph theory. Uses the Fiedler vector (eigenvector of the second-smallest eigenvalue of the interference graph Laplacian) to guide vertex ordering.

Key Insight

Traditional allocators use heuristic orderings (e.g., highest degree first). Spectral coloring uses the algebraic structure of the interference graph: the Fiedler vector provides a natural vertex ordering that minimizes edge cuts between color classes, reducing spill pressure.

Algorithm

1. Build interference graph G from live ranges
2. Compute graph Laplacian: L = D - A
   where D = degree matrix, A = adjacency matrix
3. Compute eigenvalues/eigenvectors of L
4. Extract Fiedler vector (2nd smallest eigenvalue)
5. Sort vertices by Fiedler component
6. Greedy coloring in spectral order
7. Spill uncolorable nodes

Properties

  • Complexity: O(n³) for eigenvalue computation, practical for basic blocks
  • Correctness: Produces valid colorings (no adjacent nodes share color)
  • Optimality: Not optimal, but spectral ordering reduces spill rate vs degree ordering
  • Extension: Approximate Fiedler via Lanczos iteration for large graphs
