Player 1 (Left)

WASD - Move

Q/E - Turn Musket Left/Right

R - Reload

X - Aim Down Sights

F - Fire

Player 2 (Right)

Arrow Keys - Move

I/K - Turn Musket Left/Right

L - Reload

M - Aim Down Sights

N - Fire

Reload Steps: 1. Pour Powder, 2. Ram Rod, 3. Prime Pan, 4. Cock Hammer, 5. Ready

Hold R/L to see animation. Must complete all steps before firing.
