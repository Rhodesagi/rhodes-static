LE DUEL ONE

18th Century Musket Duel - Two Players, One Computer

Player 1 (Left Screen)

WASD - Move

Q/E - Rotate Musket

X - Iron Sights (Hold)

F - Fire

R - Reload (6 stages)

Player 2 (Right Screen)

Arrow Keys - Move

Numpad 7/9 - Rotate Musket

Numpad Enter - Iron Sights (Hold)

Numpad + - Fire

Numpad 0 - Reload (6 stages)

Click to start • No crosshairs • Iron sights only • Authentic reload
