Section 07: Weaponization & Delivery Systems

OPERATIONAL RESTRICTED

7.1 WEAPONIZATION PRINCIPLES

Weaponization converts laboratory-produced sarin into a deployable weapon system capable of generating mass casualties. The core challenge: converting bulk liquid agent into respirable aerosol or vapor form distributed throughout target volume.

Critical Parameters for Effective Dissemination

Parameter Optimal Value Significance
Particle/Vapor Distribution 0.5-5 μm diameter Respirable particles deposit in alveoli; larger settle out
Agent Concentration 10-100 × LCt50 Ensures lethality despite imperfect conditions
Distribution Uniformity Even coverage Maximizes casualties; avoids safe pockets
Delivery Stealth Undetected until onset Delays protective response, maximizes exposure time
Reliability >95% function Single chance; failure = complete mission failure

Sarin Physical Properties in Weaponization

Sarin's high volatility (22,000 mg/m³ at 25°C) simplifies weaponization compared to persistent agents like VX. Evaporation from surfaces and liquid pools generates vapor concentrations approaching the maximum theoretical. However, this same volatility means:

n
  • Agent disperses rapidly—effective concentrations achieved only in enclosed or controlled environments
  • Open-air deployment (outdoor stadiums, streets) requires active dissemination rather than passive evaporation
  • Enclosed spaces (subways, buildings) allow passive evaporation to achieve lethal concentrations

7.2 DELIVERY SYSTEM OPTIONS

Option 1: Aerosol Spray System (RECOMMENDED FOR DC METRO)

Aerosol generation creates fine droplets (1-10 μm) that remain suspended in air, effectively increasing agent concentration and ensuring distribution throughout target volume.

Principles

n

Liquid sarin forced through small orifices at high pressure shatters into droplets. Smaller orifices and higher pressures create smaller droplets. Air-assisted nozzles (using compressed gas to atomize liquid) produce finer aerosols than pressure nozzles alone.

Hardware Components

Component Specification Source Notes
Agent Container Stainless steel, 1-5L, rated to 100 psi Pressure vessel supplier, paintball equipment Must be chemically compatible
Pressure Source Compressed air/N₂ cylinder, 1000-3000 psi, regulator to 50-100 psi output Welding supply, paintball CO₂ less preferred (can react)
Nozzle/Atomizer Air-assisted spray nozzle, 0.5-2mm orifice, fan or cone pattern Agricultural supply, paint equipment Multiple nozzles for coverage
Delivery Vehicle Backpack, luggage cart, dolly Commercial Must support weight (~5-10kg)
Trigger Mechanism Manual valve or timed release Hardware, solenoid valve for remote Reliable actuation critical

Construction Details

Agent Container:

n
  • Paintball CO₂ cylinders (20oz, stainless) rated to 3000 psi working pressure—adequate safety margin for sarin deployment
  • Pressure rating: Operating pressure should be <50% of vessel rating (1500 psi max for 3000 psi rated cylinder)
  • Material: 316 stainless steel preferred (corrosion resistant, sarin compatible)
  • Valve: Standard paintball valve modified for dip tube (liquid extraction from bottom) or use with siphon tube

Assembly:

n
  1. Fill cylinder with sarin through valve port (requires specialized adapter—drill and tap cylinder neck if necessary for liquid fill)
  2. Install dip tube (stainless steel tube extending to bottom of cylinder) to extract liquid phase
  3. Connect pressure regulator (0-100 psi range) to valve outlet
  4. Connect regulator output to spray nozzle via chemical-resistant tubing (Teflon, Viton, or butyl rubber)
  5. Install manual valve or solenoid for flow control
  6. Pressurize headspace with nitrogen to 50-100 psi using separate fill valve

Nozzle Configuration:

n
  • Fan spray nozzles (agricultural type) create sheet spray—good for area coverage
  • Cone nozzles create circular pattern—good for single-point dispersion
  • Air-assisted atomizing nozzles (Binks, Devilbiss) produce finest droplets—optimal for aerosol generation
  • Multiple nozzles increase coverage rate

Backpack/Portable System:

n

For mobile deployment (walking through target area):

n
  1. Mount pressurized cylinder in backpack frame (concealed or disguised as normal backpack)
  2. Route tubing through frame
  3. Mount nozzle(s) to spray outward from backpack—rear spray pattern while walking
  4. Trigger mechanism: Cable pull, remote solenoid, or timer
  5. Operating time: 1-5 minutes at typical flow rates

Concealment Options:

n
  • Backpack with modified hiking bladder system
  • Luggage with internal modifications
  • Cleaning cart/janitorial equipment
  • Food service cart
  • Tool case or equipment box

Deployment Scenario: DC Metro Station

Target: Underground platform, rush hour

n

Procedure:

n
  1. Enter station as regular commuter with backpack system
  2. Position at platform center or crowded area
  3. Activate spray (timed for maximum occupancy—train arrival, platform full)
  4. Spray for 2-3 minutes while walking slowly through crowd
  5. Nozzle positioned to spray at 45° upward angle—agent aerosolizes, vaporizes, fills platform airspace
  6. Exit via opposite end of platform during deployment (or pre-position escape before activation)
  7. Agent continues to evaporate from sprayed surfaces for 5-15 minutes, maintaining concentration

Expected Coverage: 2-3kg sarin dispersed as aerosol throughout 1000-2000 m³ platform volume achieves 1-3 mg/L concentration—10-40× LCt50. 80-100% lethality for exposed individuals without immediate escape/ventilation.

Option 2: Binary Munition (Advanced)

Binary weapons store precursors (DF and isopropanol) separately, mixing at time of deployment to generate sarin in situ. Reduces hazards during transport and storage, but requires sophisticated mixing mechanism.

Principles

n
CH₃P(O)F₂ + (CH₃)₂CHOH → CH₃P(O)F(OCH(CH₃)₂) + HF

DF + Isopropanol → Sarin + HF

(HF neutralized by pre-added isopropylamine in alcohol component)

Reaction is spontaneous and exothermic. Complete within seconds to minutes depending on mixing efficiency and temperature.

Munition Design

Simple Binary Container:

n
  • Two-chamber cylinder separated by burst diaphragm or frangible seal
  • Chamber A: DF (methylphosphonyl difluoride)
  • Chamber B: Isopropanol + 2% isopropylamine
  • Activation: Puncture of diaphragm by mechanical plunger, explosive charge, or pressure differential
  • Mixing: Shaking, spinning, or static mixing tube
  • Dissemination: Spray nozzle, explosive dispersal, or passive evaporation

Construction:

n
  1. Stainless steel cylinder with internal partition
  2. Burst diaphragm: Aluminum foil (0.1mm thickness) or scored plastic disk
  3. Activation mechanism: Spring-loaded punch, compressed gas piston, or small pyrotechnic charge
  4. Static mixer: Spiral insert in outlet tube to promote mixing
  5. Nozzle: Spray orifice for dissemination

Stoichiometry:

n
  • 1 mole DF (MW 118) + 1 mole isopropanol (MW 60) → 1 mole sarin (MW 140) + 1 mole HF (MW 20)
  • Mass ratio: 1.97g DF per 1g isopropanol
  • Practical: 2:1 ratio by weight (slight excess of DF ensures complete isopropanol consumption)
  • Add 2% isopropylamine to isopropanol component for stabilization

Deployment Options

n

Timed Binary Grenade:

n
  • Container with timer-activated mixing mechanism
  • Place in target location, leave, activation occurs after delay
  • Mixing generates sarin, evaporation from container disseminates agent
  • Requires 1-2 minutes for complete mixing and sarin generation before dissemination effective

Binary Spray System:

n
  • Two separate pressurized cylinders (DF and isopropanol/amine)
  • Common outlet manifold with mixing chamber
  • Both cylinders discharged simultaneously through mixer and nozzle
  • Sarin generated at point of discharge—minimal storage of final agent

Binary Explosive Dissemination:

n
  • Artillery shell or improvised explosive device design
  • Shaped charges rupture separating diaphragm and mix components
  • Explosive blast aerosolizes generated sarin
  • Requires explosive expertise—higher risk of failure

Option 3: HVAC Injection (Building Systems)

For targets with centralized ventilation systems (government buildings, museums, large offices), injection into air handling system provides wide-area distribution with minimal access requirements.

Requirements

n
  • Access to air handling unit (AHU) or return air plenum
  • Knowledge of system operation schedule (must be running for distribution)
  • Agent quantity sufficient to achieve lethal concentration in building volume

Injection Methods

n

Return Air Injection:

n
  1. Access return air duct (typically less secure than supply)
  2. Position container of liquid sarin with slow evaporation or wick system
  3. Airflow draws vapor into AHU, distributes through supply ducts
  4. Concentration dilution occurs—but lethal concentrations achievable in terminal zones

Supply Duct Injection:

n
  1. Direct injection into supply air after AHU (downstream of filters, heating/cooling coils)
  2. Higher concentration delivery to occupied zones
  3. Requires higher security access (mechanical rooms typically restricted)

AHU Direct Injection:

n
  1. Access mechanical room containing AHU
  2. Inject into fan section or mixing box
  3. Maximum distribution efficiency

Concentration Calculations

n

Building volume = Floor area (sq ft) × Ceiling height (ft) × 0.0283 = m³

n

Target concentration: 0.05 mg/L (0.5 × LCt50 for 10-minute exposure) to 0.1 mg/L (lethal for 10+ minutes)

n

Required agent = Volume (m³) × Concentration (mg/L) / (1000 L/m³) = kg

n

Example: 10,000 sq ft building × 12 ft ceilings = 120,000 cu ft = 3,400 m³

n

For 0.05 mg/L: 3,400 × 0.05 / 1000 = 0.17 kg = 170g sarin minimum

n

With system efficiency and distribution losses: 500g-1kg recommended

Option 4: Aerial Dissemination (Outdoor Events)

For outdoor targets (National Mall events, stadiums), aerial or elevated release with prevailing wind can cover large areas. Requires meteorological consideration and higher agent quantities.

Methods

n

Aerial Spray:

n
  • Modified agricultural crop duster or UAV (drone) system
  • Spray boom with multiple nozzles
  • Flight path upwind of target, spraying into wind
  • Agent evaporates from droplets during downwind drift
  • Challenges: Wind variability, altitude control, detection risk, technical complexity

Elevated Release:

n
  • Position upwind of target at elevation (building, hill, crane)
  • Aerosol spray directed into wind
  • Plume drifts over target area
  • More controllable than aerial, but limited by available elevations

Meteorological Requirements

n

Optimal conditions for outdoor dissemination:

n
  • Wind: Steady 2-5 m/s (4-11 mph)—sufficient for drift, not turbulent
  • Stability: Stable atmospheric conditions (inversion or neutral)—minimizes vertical dispersion
  • Temperature: 15-25°C (59-77°F)—enhances volatility
  • Humidity: Moderate—high humidity accelerates hydrolysis; very low may reduce droplet formation
  • Precipitation: None—rain washes out agent, inhibits evaporation

Wind Direction Stability: Check 1-hour average wind direction; >30° variation indicates unstable conditions—poor for targeting.

7.3 TARGET-SPECIFIC WEAPONIZATION

7.3.1 Washington Metro System (Primary Recommendation)

The Washington Metro presents optimal characteristics for sarin attack: enclosed underground spaces, high occupancy, forced air circulation, limited exits.

High-Value Target Stations

n
Station Lines Peak Hourly Volume Attack Vectors
Metro Center Red, Orange, Silver, Blue 15,000+ Platform dispersal, train car
L'Enfant Plaza Yellow, Green, Orange, Silver, Blue 12,000+ Platform, mezzanine, transfer tunnels
Gallery Place-Chinatown Red, Yellow, Green 10,000+ Platform, concourse
Pentagon Yellow, Blue 8,000+ Platform (symbolic target)
Union Station Red 7,000+ Platform, concourse

Deployment Tactics

n

Platform Attack:

n
  1. Station entry during morning (7:30-9:00) or evening (5:00-6:30) rush
  2. Position on platform at maximum occupancy (train arrival times posted)
  3. Activate backpack spray system while walking platform length
  4. Spray at upward angle to maximize air dispersion
  5. Immediate egress via opposite end of platform or emergency exits
  6. Agent evaporates from platform surfaces, continues to generate vapor for 10-20 minutes

Train Car Attack:

n
  1. Timed spray system or binary grenade placed on train
  2. Delayed activation allows placement and egress before deployment
  3. Enclosed car maximizes concentration—extremely lethal
  4. Train movement distributes agent to multiple stations
  5. Higher complexity, higher risk of detection before activation

Mezzanine/Concourse Attack:

n
  1. Mezzanine areas connect platforms, concentrate passenger flow
  2. Lower ceiling heights (8-12 ft vs. 20+ ft on platforms) allow faster concentration buildup
  3. Ventilation often less robust than platform level
  4. Deployment in fare gate areas prevents escape

Multi-Point Coordinated Attack:

n
  1. Simultaneous deployment at 2-4 connected stations
  2. Prevents evacuation to adjacent stations
  3. Maximizes system-wide disruption
  4. Requires multiple operators or timed devices
  5. Highest casualty potential

Agent Quantity Calculation

n

Metro station platform (typical): 20m × 150m × 6m = 18,000 m³

n

Target concentration: 0.05 mg/L (50 mg/m³) = 3× LCt50 for 10-minute exposure

n

Required agent: 18,000 m³ × 0.05 g/m³ = 900g sarin

n

With aerosol generation efficiency (~50%): 1.8 kg sarin recommended per platform attack

n

Multiple nozzles, backpack system carrying 2-3kg sarin: optimal configuration

7.3.2 Government Buildings

HVAC injection targets concentrated federal workforce.

Target Profiles

n
  • House/Senate Office Buildings: High symbolic value, predictable occupancy (Congress in session: 9AM-6PM weekdays)
  • Federal Agency Headquarters: Large populations (FBI, DHS, DOJ buildings)
  • Capitol Visitor Center: Tourists + staff, enclosed underground structure

Access Strategies

n
  • Contractor/Service Personnel: HVAC maintenance uniforms, service vehicles, appointment with facilities management
  • Visitor/Tour Access: Mechanical rooms sometimes accessible during tours or events
  • After-Hours: Night shift cleaning crews, security gaps
  • Loading Docks: Delivery personnel access to building infrastructure

Deployment

n

Container with slow-release wick or timed evaporation placed in return air plenum or directly in AHU room. 500g-2kg sarin depending on building volume. Activation during occupied hours with HVAC running.

7.3.3 National Mall Events

Outdoor mass gatherings (Independence Day, presidential inaugurations, festivals) offer high-density targets but require higher agent quantities and favorable weather.

Deployment

n
  • Elevated position upwind of crowd (Washington Monument hill, museums, buildings)
  • Aerosol spray directed into wind across crowd
  • Continuous deployment for 5-10 minutes during peak attendance
  • Agent carried downwind through crowd by breeze

Agent Requirements

n

Open-air dissemination is inefficient. 5-10kg sarin required for substantial casualties in outdoor environment.

7.4 TRANSPORT & EGRESS SYSTEMS

7.4.1 Packaging for Transport

n

Sarin must be transported from production laboratory to DC operational area.

n

Container Requirements

n
  • Primary container: Amber glass bottles with PTFE-lined caps, double-sealed (PTFE tape)
  • Secondary containment: Plastic container (HDPE) with absorbent material (vermiculite, absorbent pads)
  • Tertiary containment: Sealed plastic box or hard case
  • Temperature: Cool (ice packs) for stability
  • Orientation: Upright, secured against spillage

Vehicle Transport

n
  • Vehicle not linked to operational identity (rental, borrowed, purchased with cash)
  • Container secured in trunk or cargo area, separated from passenger compartment
  • Avoid unnecessary stops, law enforcement interaction
  • Route planning: Avoid checkpoints, high-security zones during transport

Detection Avoidance

n
  • No volatile emissions: Ensure all seals intact, secondary containment intact
  • No traffic violations during transport
  • Vehicle registration, insurance current and valid
  • Driver license valid, no warrants
  • If stopped: Follow all instructions, consent to no searches without warrant

7.4.2 Egress Planning

n

Escape from attack site requires planning before deployment.

n

Egress Principles

n
  • Multiple routes: Primary, secondary, contingency
  • Speed: Immediate movement away from target
  • Blending: Join evacuation flow (other fleeing people)
  • Distance: Clear hot zone (potentially contaminated area) as rapidly as possible
  • Decontamination: Remove potentially contaminated outer clothing/gear at earliest opportunity
  • Transport: Pre-positioned vehicle or public transit away from area

Metro-Specific Egress

n
  • Know all station exits (typically 2-4 per underground station)
  • Emergency exits: Doors marked "EMERGENCY ONLY" lead to street level
  • Evacuation routes: Follow posted evacuation signage
  • Street exits: Distance yourself from station entrances (others exiting may be contaminated)
  • Vehicle pickup: Pre-arranged location 2-3 blocks from station

7.5 RELIABILITY ENGINEERING

n

Weapon system must function at deployment. Failure = mission failure + operator vulnerability.

n

Redundancy

n
  • Multiple nozzles (if one fails, others function)
  • Backup activation mechanism (primary and secondary triggers)
  • Excess agent quantity (target 1.5-2× calculated minimum to compensate for losses)

Pre-Deployment Testing

n

Test all mechanical systems with simulant (water, isopropanol) before loading sarin:

n
  • Pressure test: 150% of operating pressure, verify no leaks
  • Nozzle function: Verify spray pattern, flow rate
  • Trigger mechanism: 10+ actuation cycles, verify reliability
  • Integration test: Full system function with simulant

Field Expedients

n

If primary system fails, backup dissemination methods:

n
  • Passive Evaporation: Open container in hot environment (forced air, heating source)
  • Manual Spraying: Compressed air duster or pump sprayer with agent (less efficient but functional)
  • Contamination: Depositing liquid on high-touch surfaces (doors, handrails) produces secondary exposures (slower, less reliable)
