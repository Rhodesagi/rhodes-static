MUSKETS FIRE 3

Two-Player Same-Computer Musket Duel

CONTROLS

PLAYER 1 (TOP SCREEN)

WASD - Move

Mouse - Aim

Left Click - Fire when loaded

R - Start reload

Shift - Hold breath to steady aim

PLAYER 2 (BOTTOM SCREEN)

Arrow Keys - Move

Mouse - Aim

Left Click - Fire when loaded

Enter - Start reload

Right Ctrl - Hold breath to steady aim

MUSKET MECHANICS

• Iron sights only - no crosshairs

• Full 6-stage reload process required after each shot

• Cannot move while reloading

• First to 3 hits wins
