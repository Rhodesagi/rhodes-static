THE MUSKET DUELLERS

A 2-Player Split-Screen Musket Duel

Download & Play

Download Game (ZIP)

Requires Python 3.8+ and Pygame
pip install pygame then python game.py

Iron Sight Aiming

No crosshairs. Hold shift to bring up iron sights and zoom for accuracy. Watch your ballistic drop!

Authentic Reloading

9-step reloading sequence from bite cartridge to full cock. Each step takes time - find cover!

Split Screen

Two players, one computer. Top screen for Player 1, bottom for Player 2. Keyboard controls for both.

Ballistic Physics

Projectiles have velocity and drop. Lead your shots and compensate for distance.

Controls

Player 1 (Green Coat)

WASD - Move

QE - Adjust aim

L SHIFT - Iron sights (hold)

SPACE - Fire

R - Start reload

Player 2 (Red Coat)

↑↓←→ - Move

,. - Adjust aim

R SHIFT - Iron sights (hold)

ENTER - Fire

P - Start reload

Reloading Sequence

1. Bite Cartridge
2. Prime Pan
3. Pour Powder
4. Insert Ball
5. Ram Home
6. Return Ramrod
7. Half-Cock
8. Prime Again
9. Full Cock
