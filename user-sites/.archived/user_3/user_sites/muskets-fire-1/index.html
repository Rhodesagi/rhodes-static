MUSKETS FIRE

Two-Player Split-Screen Musket Duel

Controls

Player 1 (Red Coat)

Move: WASD

Aim: Mouse

Fire: Left Click

Iron Sights: Right Click (Hold)

Reload: R (when empty)

Player 2 (Blue Coat)

Move: Arrow Keys

Aim: Right Ctrl + Arrows

Fire: Right Shift

Iron Sights: Right Alt (Hold)

Reload: Numpad 0 (when empty)

Musket Mechanics

• Iron sight aiming required for accuracy

• Full reload sequence: Powder → Patch → Ball → Ram → Prime

• Reload interrupts if you move or switch weapons

• First to 5 hits wins the duel

Authentic 18th Century Flintlock Mechanics • Split-Screen Pseudo-3D Engine
